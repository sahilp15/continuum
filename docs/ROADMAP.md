# Roadmap

## Now (MVP → closed beta)

Phases 1–9 in `IMPLEMENTATION_PLAN.md`: real tenancy on Postgres → ingestion → retrieval
hardening → connector foundation (first real connector) → remote MCP + API → extension →
governance → exceptional UI → beta.

## Beta metrics (product hypotheses, instrumented via analytics abstraction)

Time to first useful context (<10 min) · % creating a second Space · % connecting a source ·
candidate approval rate (≥50%) · bundle usefulness (≥75%) · weekly usage (60% × 3 days) ·
willingness to pay (5 of 15 design partners) · cross-Space leakage (**zero, always**).

## Next (post-beta)

Fifteen native connectors · Team plan (shared Spaces, roles, approval workflows, client
review) · automation bridges (Zapier/Make/n8n/Pipedream) · connector registry · pricing GA
(Free / Pro $19 founding / Team) behind flags until validated.

## Later (vision)

The context operating system for a person's digital life: 100+ connectors via SDK +
marketplace (manual review, security scanning, rapid revocation), agencies → lawyers →
recruiters → sales → students → researchers → developers → executives, org-level
governance, approved automations (action level 5).

## Explicit non-goals for now

Building the marketplace, >6 native connectors, billing-first development, model-assisted
preflight beyond the provider abstraction, merging external AI platforms' native memories.
